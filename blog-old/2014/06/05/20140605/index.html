function addLoginUser_listview(result){
    $("#listview_result-user li").remove();
    console.log(result[0].name);

    //$('#listview_result-user').append('<li><a>hello</a></li>').listview('refresh');
    var output='';
    $.each(result,function(index,value){
        output += '<li><a>'+value.name+'</a></li>'
    });
    $('#listview_result-user').append(output).listview('refresh');
}

peaches deploy heroku

出現一個奇怪的問題

竟然是reinitialize ?? 其實一開始我沒發現有啥問題

但是在commit之後 看git status 竟然出現之前commit的東西,這樣會把我另一個專案蓋掉! 奇怪 為什麼?

這明明是新的?

才想到 原來我這專案 是直接複製原本專案來改的 所以底下藏了同一個隱藏檔.git

先把commit的 reset掉

找到隱藏的.git

將它刪除 rm -rf .git

然後再次git init

這次就是initialzed了

git commit -m "註解"

登入heroku 並且 create新的repository 並給url

git push heroku master

成功了 !!!
