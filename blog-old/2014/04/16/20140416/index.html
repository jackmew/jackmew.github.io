又發現 回傳的是json 不是javascript

接下來要試試看 將回傳的payload 包成javascript

return new ResponseEntity<Payload>(payload, HttpStatus.OK);

回傳的會是json

我要在回傳之前將payload包起來 變成javascript

那要怎麼把payload這個java bean convert to json ?

我們用來處理json的jar是用這個

 <dependency>
            <groupId>com.fasterxml.jackson.core</groupId>
            <artifactId>jackson-core</artifactId>
            <version>2.2.3</version>
        </dependency>
        <dependency>
            <groupId>com.fasterxml.jackson.core</groupId>
            <artifactId>jackson-databind</artifactId>
            <version>2.2.3</version>
        </dependency>

http://www.journaldev.com/2324/jackson-json-processing-api-in-java-example-tutorial

www.mkyong.com

jackson jar中有個ObjectMapper 可以拿來轉換javaBean to json

mapper.writeValue(System.out,payload);

會直接將payload輸出成json到System.out(console)

String s = mapper.writeValueAsString(payload);

將javaBean 轉成json string

output s : {"status":"Y","code":null,"message":null,"items":[{"role":"S"}]}
