#express node 回憶操作流程

~/node/microblog_mongo 已經訂立好mongo連接的express project

cd ~/node/microblog_mongo

node app.js 啟動node server後就可以在browser瀏覽

mongod local mongodb才會被啟動

server debugging , node-inspector

npm install -g node-inspector

啟動 node-inspector --web-port={8081}

然後啟動node server node --debug app.js

auto refresh , supervisor

npm install -g supervisor

supervisor app.js 此時你的code有變動 儲存後 會自動refresh

可以同時啟動

supervisor --debug app.js
