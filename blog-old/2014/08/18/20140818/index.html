Java Naming and Directory Interfacae (JNDI)

realize JDBC and JNDI

link

其實這兩者之間沒有什麼關係 , JNDI 跟 Datasource比較有關係

JNDI : 為了存取 連接兩個不同service(而這service的命名方式和目錄名稱 可能會不同) , 所定義的規範.

是一個連線儲存區介面。 在執行時期，開發人員可以利用這個介面來間接存取所有 Java 資源。 JNDI 有助於解決程式碼和外部資源之間，緊密連結的問題。
