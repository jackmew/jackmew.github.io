Method Security Expression => add method level role based security to the controller.

link

先設定@EnableGlobalMethodSecurity(prePostEnabled=true) => 1. enable global method security 2. enable prePost

註解掉 原本authorize的方式 //.antMatchers(“/strategy/**”).hasRole(“ADMIN”)

然後在你要@PreAuthorize的controller加上 @PreAuthorize("hasRole('ROLE_ADMIN')")

這樣就完成basic method security expression了
