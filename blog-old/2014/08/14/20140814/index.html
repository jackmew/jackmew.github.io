Spring Boot guide

link

Spring Boot offers a fast way to build applications. It looks at your classpath and at beans you have configured, makes reasonable assumptions about what you’re missing, and adds it. With Spring Boot you can focus more on business features and less on infrastructure.

  2. homebrew

spring-boot-starter

spring-boot-starter-parent => useful maven default , dependency-management(spring-boot-starter-* can omit version)

spring-boot-starter-web => tomcat web server , Spring Boot

@EnableAutoConfiguration => tells Spring Boot to “guess” how you will want to configure Spring, based on the jar dependencies that you have added.

比如說spring boot 內含 tomcat , 就會自己去抓tomcat的需要的configuration

@configuration => 告訴spring 這個class是用來做configuration的 有兩種方法 import additional configuration classes 1. @import 2. ComponentScan

@ComponetScan => 告訴spring 額外需要import的component/configuration class 的 path(位置)

@RestController => tells Spring to render the resulting string directly back to the caller. =>combines @Controller and @ResponseBody

@Controller

mvn clean package spring-boot:runlink

cros

從最簡單的@RestController開始(spring-boot-starter這個topic 先做好了最簡單的 Restcontroller) , 變成提供resource給人call的@Controller

@ResponseBody => tells Spring MVC that it does not need to render the greeting object through a server-side view layer, but that instead that the greeting object returned is the response body, and should be written out directly. 他會告訴spring 不經過servier-side view layer(jsp , thymeleaf) , 直接傳出去

Greeting object convert to JSON => spring 內含 Jackson2 所以Spring’s MappingJackson2HttpMessageConverter 會自動轉換

convert jar to war

spring-boot-maven-plugin

  1. 原本的執行方式 : java -cp target/first-1.0-SNAPSHOT.jar org.jack.first.App
  1. 打包成可執行jar : java -jar target/jar-war-1.0-SNAPSHOT.jar

加上

<build>
  <plugins>
      <plugin>
          <groupId>org.springframework.boot</groupId>
          <artifactId>spring-boot-maven-plugin</artifactId>
      </plugin>
  </plugins>
</build>

when using the java -jar command 會找到 public static void main() 進入點去執行

  1. 打包成可執行war 在tomcat執行
