impress.js

tutorials

<body class="impress-not-supported"> 判斷瀏覽器有沒有資源impress.js

data-transition-duration="2000" 頁面轉換的時間 (ms)

data-perspective="500" 透視 (pixels) default(1000)

class="step" 會讓impress給予positione rotate scale 然後camera show them

data-x="-1000" data-y="-1500" => the center =>It will not be rotated or scaled

data-scale="4" 表示四倍大

data-rotate="90" 轉90度

class='future' 表示還沒顯示

class='present' 才顯示

class='past' 執行完

data-rotate-x, data-rotate-y, data-rotate-z(data-rotate) 繞著軸轉

var api = impress(); access to the API that controls the presentation.

api.init - 初始化 presentation

api.next() - 轉到下一頁(step)

api.prev() - 轉到上一頁(step)

api.goto(idx | id | element , [duration]) - 選定要轉到哪 並決定速度(ms)
