定義是 只有自已和1可以除它

prime number : a number divisible by only itself and 1

public static void main(String[] args) {

        System.out.println("For prime number");

        //prime definition: a number divisible by only itself and 1

        int number = 0;
        boolean isPrime = true;
        Scanner reader = new Scanner(System.in);
        number = reader.nextInt();

        for(int i=2;i<number;i++){
            if(number % i == 0){
                isPrime = false;
            }
        }
        if(isPrime){
            System.out.println(number+" is prime");
        }else{
            System.out.println(number+" is not prime");
        }

    }

所以只要2~number自己 中間的數都不能整除的 就是質數
但這還不是最佳解 只要測試2~number/2 就行了

//Recursive
    public static void permute(char[] list,int first,int last){
        int i;
        if(first == last){
            String cout ="";
            for(i = 0;i<=last;i++){
                cout +=list[i];
            }
            System.out.println(cout);
        }else{
            //置換第一個位置的值
            for(i = first;i<=last;i++){            
                swap(list,first,i);
                permute(list,first+1,last);
                swap(list,first,i);            
            }
        }

    }
    public static void swap(char[] list,int first,int i){
        char temp = list[first];
        list[first] = list[i];
        list[i]=temp;
    }
