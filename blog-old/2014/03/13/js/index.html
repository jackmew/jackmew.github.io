###Note

物件導向程式語言(Object-Oriented Language) Java , C++ , C#

1.封裝(Encapsulation)

2.繼承(Inheritance)

3.多型(Polymorphism) 好像比較少用

程式語言俱備這三個特性,就稱之為物件導向程式語言.

程式語言是一連串的演進過程,當處理一件事情的時候,不需要俱備這些特性,

但當同時處理多件事情時,你很難用直接的思考來處理事情

所以呢 OO就是只是一種設計樣式來讓我們程式設計師可以去處理多種問題,方便維護,加上新功能,才能將專案做大.

當然 你夠強 你想怎麼設計你的程式處理方式 就怎麼設計.

然而現在程式設計通常都是團隊一起來寫,所以你的設計方式也需要讓其他人能夠理解,別人才能維護,擴充你設計的程式.

在 Javascript中 變數 方法 皆是物件
物件又可以隨時新增子物件

1.封裝(Encapsulation)

Java , C++ , C# => class

JavaScript => Constructor function =>1.declare 2.new

2.繼承(Inheritance)

Java , C++ , C# => class extends

JavaScript => prototype 比較像擴充

###xxx-based
JavaScript – Prototype-based programming language :

類別事實上就是物件 => 全部都是物件

C++,C#,Java – Class-based programming language:

class => 類別是抽象資料型態

instance => class new instance

###JavaScript 101

JavaScript 101

#####Immediately-Invoked Function Expression(IIFE)

在javascipt function可以當作參數
因為在javascript中所以東西都是物件 所以function也是物件 物件自然就可以當作參數

###演進

解讀javascript =>javascript引擎

Mozilla => TraceMonkey+mozilla

Google => V8 + webkit
Safari => 應該跟Google一樣

瀏覽器 => 圖形引擎 + JavaScript引擎
JavaScript引擎 => 提供部分操作瀏覽器的功能(網路,DOM,外部事件,HTML視訊,canvas,儲存)

JavaScript:

1.ECMAScript是一種語言規範 => 數據類型 關鍵字 保留字 運算符 =>是JavaScript的骨架

2.DOM(Document Object Model) => Document=>html(content)

3.BOM(Browser Object Model) => Browser =>瀏覽器之間的作用

DOM=>每個Tag都是一個節點 節點下又有一堆屬性 可以拿來操控

BOM=>瀏覽器視窗大小 版本訊息 瀏覽歷史紀錄 cookie

#####標準化組織

JavaScript的標準化組織是ECMA

DOM的標準化組織是W3C

BOM缺乏標準

###BOM & DOM

histroy 瀏覽器的上一頁 下一頁

location 看到這頁面的host hostname port

navigator 瀏覽次數統計

screen 瀏覽的屏幕高度 單位像素

###JQuery

jQuery是一套跨瀏覽器的JavaScript library=>簡化HTML與JavaScript之間的操作

jQuery物件 <=> DOM物件 是不一樣的

jQuery物件只能用jQuery方法 <=> DOM物件只能用DOM物件的方法

jQuery物件 轉=> DOM物件
$(“#container”).get(0)

DOM物件 轉=> jQuery物件
$(element)

###JQuery 特性

jQuery有下列特色:
1.跨瀏覽器的DOM元素選擇

2.DOM巡訪與更改：支援CSS

3.Events

4.CSS操縱

5.animate

6.Ajax

7.jQuery Utility

8.jQuery plugin
