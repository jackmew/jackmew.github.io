jquery mobile

在wbase的案子中 我們是使用jquery mobile 做我們的前端開發界面

jquerymobile.com/

基本上jquey mobile 的所有Widget 都具備跨平台的特性,也就是說你用它的UI在所有的瀏覽器啊,所有的平板,手機都不會產生異常,某個CSS不work之類的

但是這也表示比較難自己刻出客制化的模樣,因為它已經定義出framework或者使用常規這樣,當你要改變原本它給你的設定時,會比較麻煩,要override某些特性,但

override的同時又要注意會不會蓋掉你原本有用到的特性,如果都不改變它給你的widget,的確是非常好用的jquery UI,而且原本給你的模樣,其實也都有美化過了,但若是

你每個地方都想要與他人不同,那就不推薦使用jquery mobile囉

jquery mobile 筆記

w3c jquery mobile

###Your own jquery mobile icon

.ui-icon-dotted:after {
    background-image: url("../icon/dotted.png");
    /* Make your icon fit */
    background-size: 30px 30px;
}

dotted是你命名這個icon的名稱

url是你圖片放置位置

size若是你的圖片大小不是正方形 ,那你可能要花點時間,才能調出不扭曲的形狀

原本jquery mobile icon是切出來方方正正的圖形

###Override

.ui-page {
    background: rgb(188,188,188) !important;
}

override你原本頁面的背景顏色

在override時都要在後面加上!important

之前也有override botton大小,但是發現這個屬性會給其他tab input使用到,所以一改,其他地方也都會改到, 非常困擾

###ui-field-contain

form

要出現平行時 需加上class=”ui-field-contain”

<div class="ui-field-contain">
    <label for="textinput-1">Text Input:</label>
    <input type="text" name="textinput-1" id="textinput-1" placeholder="Text input" value="">
</div>

然後這class在popup Widget or dialog Widget就無法產生效果,已經google到有很多人碰到這個bug

最後只好自己在dialog裡面用table 排版
