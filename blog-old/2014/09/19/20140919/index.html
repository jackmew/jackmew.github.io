mvn clean vs compile vs package vs install

所以說 只要build bat module的 最少指令應該是

~/project/tbs/svn/impl/trunk/source/web --> mvn clean install

因為web是build成 war 檔 , war 包了其他所有module jar , 所以web 每一次都必須install
