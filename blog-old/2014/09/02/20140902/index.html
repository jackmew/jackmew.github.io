cordova change icon => cordova-gen-icon

非常快速的方法

唯一的問題是 可能是版本的關係 config.xml的位置不同 , 所以當需要產生圖片的時候,

  1. 暫時將config.xml複製到www/底下

  2. 將需要的圖片 取代原本www/img/logo.png

  3. cordova platform add ios

  4. cordova-gen-icon (幫你將你的logo.png 變成各種解析度)

  5. cordova build ios

  6. cordova emulate ios 就成功了 超簡單
