決定user設定logging顯示的方式

conversion pattern : “%r [%t] %-5p %c - %m%n”

output : 176 [main] INFO org.foo.Bar - Located nearest gas station.

  1. %r => program啟動後到執行這行logger的時間(millisecond)

  2. [#t] => 誰(thread)在執行這個log request

  3. %-5p => the level of the log statement

  4. %c => the name of the logger

    • %m%n => the message of the statement
