理解問題是問什麼?

ex: 質數 一開始想到的2 3 5 7
但這樣是不對的 應該想的是定義 為什麼2 3 5 7是質數
定義是 只有自已和1可以除它

prime number : a number divisible by only itself and 1

public static void main(String[] args) {

        System.out.println("For prime number");

        //prime definition: a number divisible by only itself and 1

        int number = 0;
        boolean isPrime = true;
        Scanner reader = new Scanner(System.in);
        number = reader.nextInt();

        for(int i=2;i<number;i++){
            if(number % i == 0){
                isPrime = false;
            }
        }
        if(isPrime){
            System.out.println(number+" is prime");
        }else{
            System.out.println(number+" is not prime");
        }

    }

所以只要2~number自己 中間的數都不能整除的 就是質數
但這還不是最佳解 只要測試2~number/2 就行了
