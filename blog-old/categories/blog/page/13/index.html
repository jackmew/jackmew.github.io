mongoose-simpledb

Chevex製作的一個在express中 模組化使用mongoose的node-module:

  1. mongoose中的定義: schema -> model -> instance

Schema

var kittySchema = mongoose.Schema({ name: String });

Model

var Cat = mongoose.model('cat',kittySchema);
( var modelName = mongoose.model(‘collectionName’); )

Instance

var HelloKitty = new Cat({nam:'HelloKitty'});

通常的用法是

model -> find,findOne …

db.Cat.findOne({'name':'HelloKitty'})

Instance -> save…

db.HelloKitty.save()

  1. 上面的schema -> model -> instance定義,在mongoose-simpledb中,都定義在dbmodels內,dbmodels的每一個js都是一個model

ex:dbmodels/Kitten.js

因此schema,virtual properties,static function 都定義在裡面

要注意的是kitten.js會自動被變成大寫的Kitten.js,所以引用時也都必須是大寫開頭.

3.那要怎麼抓到這些model?

var simpledb = require('mongoose-simpledb');

var db = simpledb.init('mongodb://localhost/test');

當simpledb.init連接上資料庫後 會回傳資料庫訊息和自動去抓/dbmodels,這些東西包成db object

db.Kitten就會指到Kitten model

db.Kitten.findOne({'name.first':'mew'},function(err,koda){ if (err) return console.error(err); console.log(koda); });

最基本的tcp-server.js

var net = require('net');

var server = net.createServer(function(socket){
    socket.write('hello\n');
    socket.write('world\n');

    socket.on('data',function(data){
        socket.write(data);
    });
})

server.listen(8000);

只能自己的terminal回傳自己key在console的資料

net = require('net');

var sockets = [];

var s = net.Server(function(socket){
    sockets.push(socket);

    socket.on('data',function(d){
        for(var i = 0;i<sockets.length;i++)
            sockets[i].write(d);
    });
});

s.listen(8000);

new 一個array ,將大家的socket放在裡面

此時在另一個terminal就可以讀到共同的sockets data 然後data被write出來

在結束後 remove dead socket 

var sockets = [];

var s = net.Server(function(socket){
    sockets.push(socket);

    socket.on('data',function(d){
        for(var i = 0;i<sockets.length;i++){
            sockets[i].write(d);
        }
    });
    //remove dead socket
    socket.on('end',function(){
        var i = sockets.indexOf(socket);
        sockets.splice(i,1);
    });
});

s.listen(8000);

最後加上if (sockets[i] == socket) continue ;
來判斷若是自己的socket回應的資料 就skip掉 這樣就可以解決重複看到自己
打的文字的問題 

net = require('net');

var sockets = [];

var s = net.Server(function(socket){
    sockets.push(socket);

    socket.on('data',function(d){
        for(var i = 0;i<sockets.length;i++){
            if (sockets[i] == socket) continue;
            sockets[i].write(d);
        }
    });
    //remove dead socket
    socket.on('end',function(){
        var i = sockets.indexOf(socket);
        sockets.splice(i,1);
    });
});

s.listen(8000);

這樣就是個極小型的聊天室?!
