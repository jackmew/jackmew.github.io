Handling Exception

  1. per exception => @ResponseStatus => 自定義exception class,可以在任何時候 你throw exception 讓此class接收 .

  2. per controller => conroller based exception handling => @RequestMapping-> throw exception -> @ExceptionHanlder(xxx.class)接收

  3. globally => @ControllerAdvice => 一個exception handling 給全系統使用 => annotation driven interceptor

Unhandled exception => HTTP 500

default exception happened where to dispatch

  1. spring mvc => simpleMappingExceptionResolver manage it .
  1. spring boot => default -> /error . customize -> ErrorMVCAutoConfiguration-defaultErrorView . InternalResourceViewResolver-> Thymeleaf/JSP
