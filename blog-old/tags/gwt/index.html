JSNI (JavaScript Native Interface)

link

把一個 Java method 宣告成「native」然後用另一個語言來實做內容。在 JSNI 當中，就是用 JavaScript 來實做內容。

Overlay type

StockData extends JavaScriptObject 也就是說StockData支援Javascript object

透過JSNI method 抓到Javascript object , 再來用隱藏的方式(opaque)轉成Java Object.

綜合上述幾點

  1. 先用JSONUtils.safeEval()將JSON string 變成 Javascript objects

  1. 接著用JSNI寫法(也就是說 用Javascript) 去抓Javascript object

    / JSNI methods to get stock data.
    public final native String getSymbol() /-{ return this.symbol; }-/;
    public final native double getPrice() /-{ return this.price; }-/;
    public final native double getChange() /-{ return this.change; }-/;

  2. StockData extends JavaScriptObject => JSON -> Javascript object -> Java object

workflow

ArrayList<String> stocks 裡面會放有[‘stock1’,’stock2’]

String JSON_URL = GWT.getModuleBaseURL() + "stockPrices?q=";

GWT.getModuleBaseURL() 會自動去幫你抓 http://xxx/moduleName => http://localhost:8888/stockGwt/

url = URL.encode(url); 會組出http://localhost:8888/stockwatcher/stockPrices?q=stock1+stock2

+似乎是空白的意思?

RequestBuilder builder = new RequestBuilder(RequestBuilder.GET, url);

是一個可以組出Request object的物件 => 參數(request type , url)

builder.sendRequest是一個Asynchronous Http request(AJAX) , 是一個callback function , 當成功就會執行onResponseReceived , 失敗就會執行onError

成功時 :

updateTable(JsonUtils.>safeEval(response.getText()));

  1. response.getText() => 抓到JSON String

  2. JsonUtils.safeEval(JSON string) => JSON String 轉成 Javascript Object

  3. > => Javascript Object 轉成 Java Object
